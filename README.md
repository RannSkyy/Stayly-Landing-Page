# Stayly — Premium Bali Villa & Resort Booking Platform

## Overview

**Stayly** is a modern luxury accommodation booking platform inspired by the beauty and culture of Bali.  
Designed with a premium visual experience, responsive interactions, and a highly interactive booking flow, Stayly combines elegant UI/UX with scalable front-end architecture.

The application focuses on delivering a seamless experience for users searching for villas, resorts, and curated tropical stays while also showcasing advanced React and TypeScript implementation patterns for developers.

Stayly is built to feel immersive, dynamic, and production-ready across desktop, tablet, and mobile devices.

# 🚀 Panduan Memulai (How to Run)

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/stayly.git
```

---

## 2. Masuk ke Folder Project

```bash
cd stayly
```

---

## 3. Install Dependencies

Install seluruh package dan dependency yang dibutuhkan.

```bash
npm install
```

---

## 4. Jalankan Development Server

Menjalankan server lokal untuk development.

```bash
npm run dev
```

Setelah server berjalan, buka browser dan akses:

```bash
http://localhost:5173
```

---

## 5. Build Production

Melakukan kompilasi production-ready build.

```bash
npm run build
```

Hasil build akan tersedia di folder:

```bash
dist/
```
